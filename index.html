What We Teach

Hopscotch - Ages 4-7

With Hopscotch, there’s no typing and no syntax. The language centers around blocks that you drag and drop into place to build routines. It is perfect for kids trying to get started with programming.

*Required:
Apple Device with iOS 8.0 and above

Learn More

Scratch - Ages 7-12

With Scratch, you can program your own interactive stories, games, and animations — and share your creations with others in the online community.

*Required:
Laptop with browser capabilities

Learn More

Making Websites - Ages 12-17

Kids will think logically and explore programming with two of the easiest languages to learn, HTML and CSS. Learning these languages will ease you into learning other languages like Javascript and other languages.

*Required:
Laptop

Learn More

Mobile Apps - Ages 18+

Learn to create mobile apps for Android as well as iOS using the most simple of programming languages HTML, CSS, Javascript. With Meteor's built in project templates and easy to use plugins. You can make an app in no time.

*Required:
Laptop with Mac 10.7 and above
Laptop with Windows 7 and above
